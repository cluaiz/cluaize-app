## 🛰️ THE VISION: MODERN DESKTOP COMMUNICATION APP
Cluaize is not just another web wrapper; it is a **High-Performance Desktop App** built using Tauri, React, and Vite. Our absolute mission is to provide a premium, lightning-fast, and deeply integrated user experience with a modern UI and an optional Pixel/Retro theme toggle.

### 1. 🌍 Cross-Platform Native Experience
The system MUST extract performance from the underlying OS using Tauri's Rust backend while maintaining a flawless UI in the webview.
- **Support Matrix**: Windows, macOS, Linux desktop environments.
- **Architecture**: Rust backend for OS-level tasks, React + Vite frontend for UI.
 
### 2. 🧠 UI & Logic Separation
Cluaize must handle complex chat mechanics (like WhatsApp) without becoming bloated:
- **Feature-Driven**: Logic hooks (`useChatLogic`, Web Speech) must be completely decoupled from UI components.
- **Theming**: Premium Modern UI by default, with a strict, globally controlled CSS variable system to allow one-click toggling to a Pixelated/Retro theme.
