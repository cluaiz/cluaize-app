# 🚫 Banned Words

> ⚠️ **WARNING**: Any word from this list appearing in our code, docs, or public communication makes the project look like marketing fluff — not real engineering. Developers will dismiss us before even reading the code. These words are a reputation risk.

डेवलपर्स का एक कड़वा सच समझो—वे मार्केटिंग की हवाबाजी (Magic, Blazing Fast, Shape-Shifting) से सख्त नफरत करते हैं। जैसे ही कोई सीनियर आर्किटेक्ट रिपो के टॉप पर ऐसे शब्द देखता है, वह बिना कोड पढ़े ही प्रोजेक्ट को "Marketing Fluff" मानकर बंद कर देता है।

These words are **strictly banned** from all cluaizd repositories, documentation, LinkedIn profiles, and any public communication.

### ❌ Banned List

**Fake Grandeur / AI Hype Names:**
1. **Independent** — in any form or variation. Never write this word.
2. **OS** — cluaizd is NOT an OS. Never call it an OS.
3. **Antigravity**
4. **FlashKDA**
5. **Native Neural Foundry**
6. **Digital Consciousness** / **Living Brain** — It is a database, not a sentient being.
7. **Eureka moments** — Call it "hypothesis generation" or "graph edge inference".
8. **Magic** — Code is deterministic, there is no magic.
9. **Habit Engine** / **Aadat** — Use "Execution Affordance" or "Routing Vector".

**Silicon / Hardware Hype:**
10. **Silicon Mastery**
11. **Universal Silicon**
12. **Directly on physical silicon**
13. **Silicon-native**
14. **Quantum Emulation** — We run on GPUs/CPUs, not qubits.

**Marketing Buzzwords:**
15. **Modern**
16. **High-quality**
17. **Modern** / **Next-Level**
18. **Cutting-edge**
19. **Fast** / **Blazing speed**
20. **Innovative**
21. **Zero-latency** (unless you have benchmarks proving it)
22. **Industry-leading**
23. **State-of-the-art** (unless citing an academic paper)
24. **Makkhan** — slang, keep out of official docs

### ✅ Use Instead

- Product name: just **Cluaizd**
- Runtime/engine: **Cluaizd**
- Hardware support: **cross-platform** or **hardware-agnostic**
- Security doc title: just **Security Policy**
- Runs locally: **runs locally on user hardware**
- Brain/Intelligence: **Database Engine / Compute Node**
